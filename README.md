# mycob

Набор WDL-скриптов для высокопроизводительной обработки данных NGS-секвенирования микроорганизмов.

[Mycob](https://mycob.ru/) - сервис интерпретации результатов NGS-анализа вирусов и бактерий с использованием облачных вычислительных ресурсов. 

На апрель 2026 обработано более 90 тыс. образцов. Анализ загруженных файлов производится в российских дата-центрах (платформа Yandex Cloud).
